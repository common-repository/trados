== Description ==
Translate any WordPress blog with Trados, it comes with a powerful translation management system that allows you to translate your content.

<h4>How Trados is working?</h4>
* We are using (https://www.convertizer.fr/) as translation platform to offer you the best service.

<h4>Your Privacy</h4>
* By using Tradosyou agree that 'YOUR EMAIL & YOUR BLOG NAME' will be sent to our agents so we can offer you the best of our knowledge.
* If you have any questions about our Privacy, please contact us.

== Installation == 
* In your Admin, go to menu Plugins > Add
* Search for Trados
* Click to install
* Activate the plugin 
* Start using Trados to translate your WordPress blog, just follow the [ Text collecting ] link from Trados dashboard
* Collect words, then hit Save to Trados button
* From your dashboard, hit [ Translate button ]
